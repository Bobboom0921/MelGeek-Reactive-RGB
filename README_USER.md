# MelGeek Reactive RGB 最终用户说明

版本：v2.1.2

这是非官方工具，与 MelGeek、Microsoft 或其他提到的第三方厂商没有从属、赞助、背书或官方支持关系。产品名和商标归各自权利人所有。

最终用户入口：`MelGeekReactiveRGB.exe`

## 特性

- 基于 WebView2 (Edge) + HTML/CSS 的现代桌面控制面板。
- 无 CMD 控制台窗口。
- 主控制面板支持最小化到系统托盘。
- 托盘右键可显示窗口、启动灯效、停止灯效、退出。
- 优先使用 native HID 读取压力数据。
- 音频来源是系统播放声音（WASAPI loopback），不是麦克风。
- 支持 12 种灯效：静态、呼吸、彩虹、涟漪、压力热力、音频氛围、综合响应、打字机、星空、波浪、追逐、渐变。
- 高级模式：字符区 / 背板区 / 侧边区可独立配置灯效和混合模式。
- 键盘预览支持日间、夜间和跟随系统主题。

## 首次使用

1. 双击 `MelGeekReactiveRGB.exe`。
2. 选择灯效并调整参数。
3. 如果使用压力灯效，按界面提示连接键盘。
4. 如果使用音频灯效，请先通过目标输出设备播放音乐或其他声音。
5. 点击右上角关闭按钮会将主窗口缩到系统托盘。要彻底退出，请右键托盘图标并选择退出。

- 本软件按原样提供，不提供任何担保。使用 HID、灯光控制、音频 loopback 或未签名 EXE 时，请自行承担风险。
